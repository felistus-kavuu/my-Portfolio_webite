About the project

Project summary

I worked on this project during my three month internship at Brainverse Technologies.I worked in the embedded systems department. We were working on a smart farming project that would allow farmers to remotely water their farms using their phones

What I did

I was in-charge of ensuring that the hardware(DHT sensor and moisture sensor) were sending real time data to the server so that farmers would have real-time data sent to their phones. We worked in a team of three and successfully presented the project prototype to a client.

Technologies/Languages used

Arduino(ESP8266), Firebase, C/C++

Personal contribution

  • Presented our prototype to the client.
  • Updated the sensor code by implementing bufferedOut functionality instead of serial printing which improved data delivery to Firebase by over 50%
